  • There are 40 transit stations on the grid, 10 each of RED, BLUE, YELLOW and GREEN color marked 1 to 10. These stations generate random PAX load at 4 game hours interval.
  • The game clock and game money counters are placed at the bottom of the screen.
  • The lower left icon toggles 'Build' mode. While in 'Build' mode, click/tap on two horizontal/vertical grid blocks on the same grid line to lay the horizontal/vertical track segment between the blocks.
  • The order in which the grid blocks are clicked/tapped determines the direction of the track. 'Junctions' are created at the extremities of the track segments. 'Junctions' allows vehicle movement in all direction. Each station is also a 'Junction'.
  • To remove an existing track segment between 2 grid blocks, click the blocks in the same order as that of the direction of the existing track segment.
  • To reverse an existing track segment between 2 grid blocks, click the blocks in the opposite order as that of the direction of the existing track segment.
  • To create a new 'Junction' on a grid block, click the block twice. To remove an existing 'Junction' from a grid block, click the 'Junction' block twice.
  • Mixing segment types between 2 will make the path between the isolated.
  • Exit out of 'Build Mode' to resume the game cycle.
  • Clicking a station will open the 'Information' box. The 'Info' tab lists the PAX count at the station waiting to move to the 39 other stations.
  • The 'Vehicles' tab lists all the vehicles plying on the grid. The 'Info' icon of each of the vehicles on the list opens a 'Vehicle Information' box which allows updating 'Route' of a vehicle.
  • The 'Route' of a vehicle is auto-reversing, i.e. forward journey 'Route' will be auto reversed for the return journey.
  • A vehicle by default waits 10 game minutes at each 'Stop'. This can be changed to make the vehicle 'wait till full' at desired 'Stop' from the 'Vehicle Information' box.
  • To make a 'Route' update of a moving vehicle, first 'Pause' the vehicle from 'Vehicle Information' box. Once vehicle is paused, a 'Stop' can be added/removed. Vehicle 'Origin' stop cannot be removed.
  • For any traffic gridlock, click the 'SOS' button on the 'Vehicle Information' box to clear the traffic congestion (if atomatically possible).
